Experiencia

Desarrollador Frontend y Backend en Proyecto Platzi vídeo

Febrero del 2020 - Actualidad

Platzi video es una aplicación web donde puedes reproducir tus vídeos favoritos. En está aplicación se desarrolló desde la maquetación con HTML y CSS de las diferentes vistas, se agregó interactividad con la librería favorita de la casa React. Se crearon las APIs de la aplicación con la librería de Node JS Express donde se llevó a cabo la arquitectura Clean Architecture. Y finalmente se está integrando el Backend y el Frontend.

Desarrollador Frontend y Backend en Wawa

Marzo del 2019 - Julio del 2019

Wawa es una red social para donde puedes registrarlas para poder hacer un reporte de desaparición y adopción o hasta tú podrás adoptar. Esta aplicación tiene un feed para subir contenido multimedia dónde puedes ver consejos para educar tus mascotas o simplemente ver contenido divertido de mascotas. Desarrollador Frontend y Backend en Yelp Camp

Junio del 2019 - Agosto del 2019

Yelp Camp es un proyecto personal que sirve para promocionar campamentos y mostrar su localización con la API de Google para programar una visita al aire libre. Este fue desarrollado con la librería Express y vistas tradicionales con Ejs. También se usó la API de Cloudinary para guardar los archivos.

Desarrollador Software y Hardware en Alarm-ITA

Septiembre del 2018 - Diciembre del 2018

Alarm-Ita es un proyecto escolar que se realizó con el objetivo de hacer una alarma remota que se puede encender y apagar desde un navegador web y quedará registro de quién la encendió o apagó. Si esta se llegase a activar mientras está encendida sonaría un buzzer y una cámara te empezaría a grabar y guardaría el video en una Raspberry. Se desarrolló un software con Node JS para las funciones del Hardware y Socket.IO para comunicarse con el servidor. Y el servidor y las vistas se hicierón con Express y Ejs.
